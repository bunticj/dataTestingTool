# Data validator
QA Tool for http requests

## Global dependencies

```
node.js
npm
```
#### Project setup
```
add env variables in .env file in root folder:
MONGO_USER=MongoDBUserName
MONGO_PW=MongoDBPassword
MONGO_DB=MongoDatabaseName
JWT_KEY=yourSecretStringKey
```

```
npm install
```

#### Project run
```
npm run start
```
